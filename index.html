C++ 소켓 HTTP 서버에 오신 것을 환영합니다

이 페이지는 서버가 디스크의 index.html 파일을 실제로 읽어 전송한 것입니다. (/ 또는 /index.html로 접속 시 표시됩니다.)

제공하는 API

  • GET /users → 전체 회원 목록 조회 (200)
  • GET /users/{id} → 특정 회원 조회 (있으면 200, 없으면 404)
  • POST /users → 회원 생성 (201, Location 헤더로 위치 안내)
  • PUT /users/{id} → 회원 수정 (200 / 없으면 404)
  • DELETE /users/{id} → 회원 삭제 (200 / 없으면 404)

모든 오류 응답(400·403·404·405·413·500)은 실제 서버 상태를 확인해 반환됩니다.
